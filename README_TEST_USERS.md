# 测试用户账号

## 快速开始

### 方法一：使用脚本创建（推荐）

运行以下命令创建测试用户：

```bash
npx tsx scripts/create-test-users.ts
```

### 方法二：通过注册接口创建

访问注册页面或使用 API：

```bash
curl -X POST http://localhost:3000/api/auth/register \
  -H "Content-Type: application/json" \
  -d '{
    "name": "测试用户",
    "email": "test@example.com",
    "password": "123456"
  }'
```

## 使用说明

1. **登录测试**: 使用上述任意账号登录系统
2. **功能测试**: 不同账号具有不同的权限和积分，可用于测试各种功能
3. **数据测试**: 这些账号已关联一些示例词条和评论数据

## 注意事项

- 所有测试账号的密码都是 `123456`，**仅用于开发测试**
- 生产环境请务必修改所有默认密码
- 如果用户已存在，脚本会跳过创建
- 可以通过修改 `scripts/create-test-users.ts` 添加更多测试账号

## 重置测试数据

如果需要重置测试数据，可以：

1. 删除数据库中的用户数据
2. 重新运行创建脚本

```sql
-- 警告：这会删除所有用户数据
DELETE FROM users WHERE email LIKE '%@gaoneng.wiki';
```

然后重新运行：
```bash
npx tsx scripts/create-test-users.ts
```
